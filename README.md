# IIR_orientation_Django
## How to run
1. run
```
docker-compose build
docker-compose up
```
2. remove container
```
docker-compose down
```
3. index
```
127.0.0.1:8000/index
```
4. git clone for Windows
```
git clone https://github.com/lingxuanying/IIR_orientation_Django --config core.autocrlf=input
```
